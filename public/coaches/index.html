Section 1

How to Use This Playbook

Three layers — read top to bottom for any session you're planning.

01

Pick the Theme

Open the Weekly Skill Plan for your age band and find the week's theme. Every drill that day reinforces it. One theme, one focus.

02

Build the Session

Use the sample sessions as scaffolding: warm-up, three skill stations, then scrimmage. Pick stations from the drill library that fit the theme.

03

Coach the Card

Every drill card has setup, coaching points, progressions, and common mistakes. Read it before practice. Run it the same way every coach would.
