# qmtk Soccer — qmtk.org

Elite youth soccer coaching in Andover & Reading, MA.

## Structure
- `public/index.html` — the entire website (HTML + CSS + JS, single file)
- `public/` — all images, logo, and resume PDF
- `vercel.json` — Vercel config (output directory: public)

## Deploying
This repo is connected to Vercel. Every push to `main` automatically deploys to [qmtk.org](https://qmtk.org).
